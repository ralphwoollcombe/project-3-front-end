# Description
- This project is a MERN Stack CRUD application built using MongoDB, Express, React, and Node.js.
- The app allows users to create and manage personal travel adventures around the world.
- Users can explore countries through an interactive map and document their experiences.
- The project focuses on full CRUD functionality, authentication, authorization, and full-stack integration.

# How-To-Use
- Users can sign up by creating a new account.
- Registered users can sign in securely.
- After signing in, users are redirected to the MAP page.
- Users can explore continents and select countries.
- Each country shows a list of adventures.
- Users can create a new adventure using a form.
- Adventures can be viewed in a detailed show page.
- Users can edit or delete only their own adventures.
- Users can access "Your Adventures" from the navigation bar.

# Features
- JWT-based user authentication.
- Protected routes for authenticated users only.
- Full CRUD operations for adventures.
- Country-based adventure organization.
- Interactive world map navigation.
- External API integration for country data.
- Personal dashboard for user adventures.
- Authorization to prevent editing other users' data.
- Structured storytelling sections for each adventure.
- Responsive and consistent UI.
